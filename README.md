# Usage
Getting the project.
```sh
$ git clone https://github.com/anirudhbhashyam/Tic-Tac-Toe
```

Installing `SDL2`.
```bash
$ sudo apt install libsdl2-dev
```
```bash
$ brew install SDL2
```
Building the project (assuming you are at the root of the project).
```sh
$ bash pack.sh build
$ ./app
```

# Dependencies
- `SDL2`
- `g++`
